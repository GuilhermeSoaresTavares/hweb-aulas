Códigos Úteis Git

git init: // Iniciar um repositório

git remote add origin : //Vincular uma pasta do Github a este repositório/arquivo aberto

git add: //Reconhece todas as alterações feitas e adiciona para preparar para salvar

git commit -m: Ta tudo pronto pra mandar pra nuvem

git push -u origin master: Primeiro push, subir alterações no repositório

git push origin "Nome da branch": Enviar mudanças para ser salva no repositório/Branch
