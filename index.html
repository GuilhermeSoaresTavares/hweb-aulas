Códigos Úteis Git

git init: Iniciar um repositório

git remote add origin "link do Github": Vincular uma pasta do Github a este repositório/arquivo aberto

git add "arquivo" : Reconhece todas as alterações feitas e adiciona para preparar para salvar

git commit -m "Nome do Commit - Obrigatório" : Ta tudo pronto pra mandar pra nuvem

git push -u origin master: Prim origineiro push, subir alterações no repositório

git push origin "Nome da branch": Enviar mudanças para ser salva no repositório/Branch

git checkout -b "Nome para criar a Branch": Mudar entre branchs criando uma nova branch

git checkout "Nome da Branch": Mudar entre Branch/Tarefas

git clone "Link do repositório": Clonar pasta na máquina

git status: Saber o que foi modificado
